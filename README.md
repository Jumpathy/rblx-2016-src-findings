# rblx-2016-src-findings

This repo will consist of my findings of issues I had and their solutions within 2016 Roblox's source code, as well as rundowns of how the components work, from RobloxStudio to WindowsClient to RCCService. 

Also included:

- WindowsClient required endpoints to run & play games
